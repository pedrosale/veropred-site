Simulação Cognitiva • Attention

Quando contexto vira foco.

Attention permite que um modelo selecione quais partes da informação importam mais em cada momento. Em vez de tratar tudo com o mesmo peso, o sistema redistribui foco.

Função cognitiva

Focar continuamente nas partes mais relevantes de uma informação para compreender corretamente o contexto.

Intuição

Em uma sequência longa, nem todas as palavras importam igualmente. Algumas carregam relações decisivas para interpretar o significado.

Forma conceitual

contexto → pesos → foco

Attention

Attention ajuda o modelo a conectar partes relevantes de uma sequência, especialmente quando a relação entre os elementos não está lado a lado.

Ideia central:
entender uma palavra pode exigir recuperar outra informação distante no contexto.

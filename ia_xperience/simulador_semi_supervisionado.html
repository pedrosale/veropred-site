Simulação Cognitiva • Semi-Supervisionado

Quando poucos rótulos iluminam muitos dados.

Aprendizado semi-supervisionado combina uma pequena base rotulada com uma grande base sem rótulo. O modelo usa os exemplos confirmados para estimar rótulos confiáveis no restante dos dados.

Função cognitiva

Expandir aprendizado quando existem poucos exemplos corrigidos e muitos dados ainda não classificados.

Intuição

Como aprender com poucas respostas confirmadas e usar essa base para interpretar muitos casos parecidos.

Forma conceitual

poucos rótulos → confiança → pseudo-rótulos

Conceito ativo

Aprendizado Semi-Supervisionado

O modelo começa com poucos exemplos rotulados. Depois usa a estrutura dos dados sem rótulo e suas próprias previsões confiáveis para ampliar o aprendizado.

Ideia central:
poucos rótulos humanos orientam a interpretação de muitos dados não rotulados.

Simulação Cognitiva • Autoregressivo

Quando uma resposta nasce em sequência.

Modelos autoregressivos geram conteúdo passo a passo. Cada novo token produzido passa a fazer parte do contexto usado para prever o próximo.

Função cognitiva

Prever o próximo token, palavra ou evento a partir do contexto acumulado.

Intuição

Como escrever uma frase olhando sempre para o que já foi dito antes, escolhendo a próxima parte mais provável.

Forma conceitual

contexto → próximo token → novo contexto

Geração Token por Token

Escolha um cenário e avance a geração. A simulação mostra como cada token altera o contexto e influencia a próxima previsão.

Conceito ativo

Aprendizado Autoregressivo

A resposta não aparece inteira de uma vez. Ela nasce em sequência: cada token gerado entra no contexto e influencia a próxima decisão.

Ideia central:
gerar é repetir a pergunta: “dado tudo que veio antes, qual é o próximo token mais provável?”
